圖片 Images

全部經由類別來設定，選擇加入響應式與增加輕巧樣式之文件和範例。

  • 無障礙 圖片必須要擁有 alt 屬性
  • 優化 圖片有 width 及 height 屬性可加速 render 速度。

基本 Basic

圖說 Diagram

使用 figure.img-figure 及 figcaption ，樣式已設定為置中且增加 <figcaption> 的符號。

縮圖 Thumbnails

可以使用 .img-thumbnail 讓圖片有1px邊框的顯示。

響應式圖片 Responsive images

Bootstrap 中的圖片可利用 .img-fluid 設定為響應式。

背景圖片 Background Images

可以透過 .img-setbg 將圖片設定為上層元素的背景，上層元素可使用 .setbg-cover, .setbg-contain 設定 background-size，.setbg-top 及 .setbg-center 設定 background-position，.text-hide 設定文字不顯示。
