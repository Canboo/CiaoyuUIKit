行為按鈕 Events buttons

在網頁應用上，部份的按鈕在點擊後需要觸發特定行為，可使用特定的樣式以進行套用。

確認 Confirm

按鈕點擊後需要顯示確認是否繼續動作，可使用 .btn-confirm 。如果需要指定詢問文字內容請在標籤上使用 data-confirm 屬性。

Class Preview
.btn-confirm 超連結套用按鈕範例

鎖定 Disabled

按鈕點擊後要讓按鈕無法再次被點擊，可使用 .btn-disable 。

Class Preview
.btn-disable 超連結套用按鈕範例

狀態 Status

在按鈕加入 disabled 或在 a 標籤加入 .active 或 .disabled 來表示按鈕狀態。

Class Preview
disabled
a.active 啟用狀態

* a 標籤才需要使用，各瀏覽器已針對 button 有預設按壓效果，沒必要再添加額外的 Class。
a.disabled 停用狀態

* a 標籤不支援 disabled 屬性，所以必須要額外添加 .disabled 使其看起來被禁用，並請附加 aria-disabled="true" 及 tabindex="-1" 避免被選取或套用 Focus。

倒數計時 Countdown

加入 .btn-countdown 及搭配 data-num="10" 設定秒數，以便達到倒數計時的效果。
若有修改倒數文字的需求請搭配 data-text="顯示文字"
