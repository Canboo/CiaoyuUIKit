單複選 Check/Radio

表單控制項 checkbox / radiobox 樣式和範例。

  • 無障礙 各欄位必須要擁有 id 屬性，並有相對應的 <label for="id"> 對應
  • 無障礙 如果欄位有描述文字，請使用 aria-describedby 屬性指向特定 id aria-describedby說明

複選 Checkbox

由許多的 type="checkbox" 所組成，相同群組應有相同的 name 屬性。

jQuery 取值使用
$('input:checkbox:checked[name="自定義"]').map(function() {
    return $(this).val();
}).get();

單選 Radio

由許多的 type="radio" 所組成，相同群組應有相同的 name 屬性。

jQuery 取值使用 $('input:radio:checked[name="自定義"]').val();

顏色 Colors

加上 .form-check-* 的Class使控制項看起來更亮麗。

禁用 Disable

加上 disabled 的屬性防止使用者操作並使它看起來更淡。
